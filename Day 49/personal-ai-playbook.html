What is the Personal AI Playbook?

A single-file, offline tool for turning the way you already use AI into reusable systems — instead of retyping context every time.

Dashboard

Your home base — quick stats, category shortcuts, and your favorited workflows.

Workflows

Ready-made, editable prompt templates organized by task (bug triage, PRDs, code review, brainstorming). Each has fill-in variables, a live preview, best practices, and a one-click copy button.

Prompt Builder

Assemble a prompt from labeled blocks — Role, Objective, Context, Constraints, Reasoning Strategy, Output Format, Tone, Examples, Quality Checks — with a live preview of the exact text you'll copy. Every block explains what it does before and after you add it.

Loop Builder

What's a "loop prompt"? It's a prompt that tells the AI to check its own output against criteria you define, revise it, and repeat — instead of answering once. You define the Goal, Evaluation Criteria, Improvement Strategy, Stop Conditions, and Safety Rules; the builder assembles them into one instruction block.

My Library

Every workflow you've saved, edited, duplicated, or favorited — searchable, filterable, and exportable as a JSON file you can back up or move to another browser.

Where is my data stored?

Entirely in this browser's local storage. Closing the tab won't lose it, but clearing browser data will — use Export regularly if you want a backup.
