Agent responsibilities

Planner — reads company, role, and JD focus once, and drafts the single interview question this session will coach an answer for.
Executor — writes the first draft of the candidate's answer to that question.
Evaluator — scores the current draft 0-100 against a rubric (relevance, structure, specificity, confidence) and explains the score. Runs fresh every round on the latest draft.
Critic — reads the same draft and evaluation, and gives pointed, concrete critique the Improver can act on.
Improver — rewrites the answer using the prior evaluation + critique. Its output becomes next round's draft.
Memory Manager — appends each round's score, critique, and draft delta to the running history so later rounds have full context.
Final Reviewer — fires only after the stop-check ends the loop; writes the final assessment and names the exact stop reason.
Each round: Evaluator → Critic → Improver, then the stop-check runs (plateau → threshold → hard cap, in that order) before deciding whether to loop back to Evaluator or branch to Final Reviewer.
